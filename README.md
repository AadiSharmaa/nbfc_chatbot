Navigating digital applications for financial products is often unnecessarily complicated. Conventional NBFC chatbots rely on rigid decision trees, meaning any unexpected user response or out-of-order question can cause the system to freeze or reset the conversation. Consequently, the financial industry suffers from high application abandonment rates. Addressing this limitation was the primary objective of our capstone project, developed collaboratively by Shivansh Kashyap, Rohit Shukla, and Aadi Sharma.
We designed BriskAI to replace inflexible legacy bots with a dynamic, agent-based framework that manages loan processing through smooth handoffs between dedicated AI modules.  By leveraging LangGraph for state orchestration and Groq's LLaMA 3 models for rapid natural language processing, the platform intelligently adapts to shifting user intents. BriskAI seamlessly manages diverse tasks—from clarifying terms and processing identity documents to negotiating loan parameters—without losing conversation context or violating strict industry regulations.
